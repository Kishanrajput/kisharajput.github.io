Scientific Optimization Control Toolkit (SOCT) is an open-source Python framework built on OpenAI Gymnasium for designing and deploying reinforcement learning agents for scientific and engineering optimization and control applications. It provides modular components including agents, environments, models, buffers, and utilities for developing ML-driven solutions for particle accelerators and other scientific systems.
